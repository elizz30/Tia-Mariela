Plan de Estudio - Pasantía Software Engineer en TradeStation

Duración: 12 semanas (3 meses). Objetivo: dominar C#, SQL y saber un poco de Python y fundamentos de nube para aplicar a la pasantía.

Mes 1 – Fundamentos sólidos

Semana 1: C# básico

  • Curso: C# Fundamentals for Beginners – Mosh
  • Documentación: Microsoft Learn – Introducción a C#

Semana 2: Programación orientada a objetos

  • Curso: OOP en C# – Udemy
  • Proyecto: sistema de biblioteca con clases.

Semana 3: SQL básico

  • Tutorial: SQL Tutorial – W3Schools
  • Práctica: instalar SQL Server Express y hacer consultas básicas.

Semana 4: C# + SQL

  • Tutorial: Conexión C# con SQL
  • Proyecto: app de inventario con base de datos.

Mes 2 – Aplicaciones prácticas

Semana 5: C# avanzado

  • LINQ: Tutorial LINQ – Microsoft
  • Práctica: filtrar listas con LINQ.

Semana 6: Python básico

  • Curso: Python for Beginners – Microsoft
  • Práctica: scripts para leer archivos CSV.

Semana 7: Automatización con Python

  • Libro: Automate the Boring Stuff
  • Proyecto: script que renombre archivos automáticamente.

Semana 8: Testing en C#

  • Tutorial: Unit Testing con NUnit
  • Práctica: pruebas unitarias para tu app de inventario.

Mes 3 – Preparación profesional

Semana 9: APIs en C#

  • Tutorial: ASP.NET Core Web API
  • Proyecto: API REST para tu app de inventario.

Semana 10: Fundamentos de nube

  • Curso: AWS Cloud Practitioner Essentials
  • Alternativa: Microsoft Azure Fundamentals

Semana 11: Proyecto integrador

  • Combina C#, SQL y API REST.
  • Añade un script en Python para automatizar tareas.

Semana 12: GitHub y presentación

  • Tutorial: GitHub for Beginners
  • Sube tu proyecto con README.md profesional.

Plan creado para preparar la pasantía en TradeStation Costa Rica. Autor: Elías.
