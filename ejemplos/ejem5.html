SQL (Structured Query Language o Lenguaje de Consulta Estructurado) es el lenguaje estándar para gestionar, manipular y recuperar información almacenada en bases de datos relacionales. Permite crear, actualizar, eliminar y consultar datos estructurados en tablas mediante comandos como SELECT, INSERT o DELETE, siendo fundamental en el análisis de datos y administración de sistemas.
