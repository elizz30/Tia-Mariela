Efímero: Que tiene una duración muy corta o pasajera. Se utiliza mucho para describir la belleza de las flores, un atardecer o sentimientos que se desvanecen rápido.

Petricor: Es el nombre que recibe el aroma particular que desprende la tierra seca cuando cae la lluvia. Es esa sensación olfativa tan característica del campo tras una tormenta.

Resiliencia: La capacidad que tiene una persona para superar circunstancias traumáticas o adaptarse positivamente frente a la adversidad. Es la fuerza interna para "rebotar" después de un golpe
