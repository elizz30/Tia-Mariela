Lenguajes de programación

C
Es un lenguaje estructurado; tiene estructuras de control y tipos de datos estructurados definidos por el programador
a partir de los tipos atómicos típicos y mediante arreglos, estructuras, uniones y punteros, incluidos los que apuntan a una función.
C#
C# (pronunciado "C Sharp") es un lenguaje de programación moderno, orientado a objetos y con seguridad de tipos,
desarrollado por Microsoft. Funciona principalmente sobre la plataforma .NET, permitiendo crear aplicaciones versátiles como software de escritorio,
servicios web, aplicaciones móviles y videojuegos, destacando por su sintaxis clara y multiplataforma.
JavaScript
JavaScript (JS) es un lenguaje de programación interpretado de alto nivel, utilizado principalmente para crear sitios
web interactivos y dinámicos. Actúa en el lado del cliente (navegador) y servidor, permitiendo actualizar contenido, controlar multimedia y animar
imágenes sin recargar la página. Forma parte de las tres tecnologías estándar de la web junto a HTML y CSS.
