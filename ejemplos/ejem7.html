Tipos de datos en MYSQL

varchar

se usa para almacenar cadenas de caracteres. Una cadena es una secuencia de caracteres. se coloca entre comillas (simples): 'Hola'.
El tipo "varchar" define una cadena de longitus variable en la cual determianos el maximo de caracteres. Puede guardar hasta 255 caracteres. Para almacenar cadenas hasta de 30 caracteres definimos un campo de tipo varchar(30)
